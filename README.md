# TCH1001-paint-verdadero
#Triángulo y Rectángulo
El codigo original proviene de free games Python (http://www.grantjenks.com/docs/freegames/). el juego origianl se llama paint y consiste en dibujar cin el mouse lineas de coleres.

En la función rectangle, lo que hicimos fue generar una secuencia de cuatro líneas, dos pares paralelos con mismos tamaños entre cada elemento del par, pero diferente comparado al otro par, posteriormente, entre cada línea trazada marcamos una rotación de 90 grados, tres veces hasta que se lograra cerrar.

En la función triangle, se realizó una función for en donde se tomaron en cuenta los tres lados del triángulo, y se realizó un cambio de 120 grados para poder cerrarlo.

En la funcion circle, se agrgaron 6 lineas para definir el radio del circulo dado por los puntos señalados por el mouse y con es ragio se genera el ciculo sobre el primer punto señalado. 

Se utlizaron la bibliotecas de Math y Turlte.
